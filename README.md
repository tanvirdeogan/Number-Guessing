# Number Guessing Game
The Number Guessing Game is a simple command-line application developed in Python. The program generates a random number between 1 and 100, and the user has to guess the correct number within a maximum of 10 attempts. After each incorrect guess, the program provides a hint indicating whether the guess is too high or too low. The game ends when the user guesses the correct number or uses all available attempts.

## Project Objective
The objective of this project is to develop a basic interactive game using Python. It helps beginners understand the concepts of loops, conditional statements, variables, user input, comparison operators, and the use of Python's built-in `random` module.

## Features
- Generates a random number between 1 and 100.
- Allows the user a maximum of 10 attempts.
- Displays "Too High" if the guessed number is greater than the correct number.
- Displays "Too Low" if the guessed number is less than the correct number.
- Shows the number of attempts remaining after each incorrect guess.
- Displays a congratulatory message when the user guesses correctly.
- Displays "Game Over" when all attempts are exhausted.
